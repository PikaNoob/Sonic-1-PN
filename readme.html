S1 Hacking Studio

Version 1.1, Built 11/24/2009

Program by vladikcomper

S1 Hacking Studio - is a packge to create Sonic the Hedgehog 1 hacks. ROMs are built from disassembly. S1 Hacking studio performs easy game tweaking, it features SonED2 for level editing.

Warning

  • Program path's leenght must be less than about 48 symbols.
  • This is not program for editing S1 ROMs, this program lets you build ROMs from disassembled code (it located in SourceCode folder)
  • Disassembly that S1 Hacking Studio uses is based on Hivebrain's Disassembly, but has some modifications. You shouldn't replace "sonic1.asm" file with your own. But feel free to edit source code.

How to set up program

Yes, to work with S1 Hacking Studio, you should store it a little. First of all, extract it from archive, if you haven't done it yet. Program path souldn't be shorter than 48 symbols. I don't know why, but old ASM compiler will shout if the path is too long.

Now you have to choose the location where your ROM will be writen. Choose "Compilation" tab and press "Browse" button.

Warning! If file you've choosen is already exist, it'll be overwriten, so be careful.

That's all. Now press "Save", then "Compile". Open output ROM in your emulator, it should look like original Sonic 1 ROM. If you have some problemms, read FAQ.

FAQ

  • When I try to compile a ROM, I get a critical error, which looks something like this: "The instruction at 00401dd2 referenced memory at 003e27f8".
    I guess, program path is too long for an old ASM compiler. Move program to make path to "SourceCode" folder less than 48 symbols (though I don't really know about the leength).
     
  • I can't play my game in emulator! It shows nothing.
    I guess it's caused by compilation error. You should exactly know its description. So go to "Compilation" tab, and switch off "Close compilator window when compilation is done" option, then press "Save" and "Compile". See what compiler writes.
    If the error is "Branch Offset Out of Index" or "Branch is out of range", then you should go to "Texts > Zone names" and make some names shorter.
     
  • I got a red screen.
    Hey, you're lying :) I've fixed it.
